tmux 常用命令

Posted on 2017-01-04 | Edited on 2018-10-05 | In tmux |

一点说明

如果一件事情每次都需要花费你很多时间，要么你记住它，要么你放弃它。对于 tmux 的快捷命令，老是记不住，但由于其在 ssh 和记录工作状态等优点，我选择记住他。

PS: 大家注意看这里的重点，由于原本的 <prefix>=ctrl+b 太难按了，所以我修改成了 <prefix>=ctrl+a 。如果你发现命令没反应，请确定环境里面的 <prefix> 是什么键

概念

session 输入 tmux 命令后，打开的连接就是 session

window 一个 session 可以打开多个窗口

pane 一个 window 又可以有多个 pane

看着是有点复杂，多用用就好了

session

创建一个session

如果你在终端一直输入 tmux ，那么每次打开都会创建一个新的 session ，怎么知道是新创建的呢？进入 tmux 后，输入 <prefix> s ，我把 <prefix> 修改成了 ctrl+a / ^a ，所以整个命令序列是 ctrl+a s ，注意有个空格，代表输入 s 时不用按住 ctrl 键。回车后可以看到有多个窗口，通过 jk 键选择不同的窗口。

打开一个已经存在的session

tmux 一个很有用的功能就是能够记住你的工作状态，比如今天晚上在这个状态下没有完成的工作，明天打开终端直接恢复出来，便于延续思路。那么如何操作呢？首先要学会

  1. 创建一个自己命名的 session
  1. 对已经存在的 session 修改名字

好了，进入正题，如何打开一个已命名的 session 呢，输入 tmux a -t sessionName

清理session

开始的时候特别困惑，又是 session ，又是 window ，还有 pane 的，把整个 tmux 弄得乱七八糟的，想恢复成最开始的状态，又不知道如何做。告诉你，可以输入 <prefix> :kill-server ，回车过后一切都安静了

保存session（暂时离开session回到终端）

有时需要我返回终端操作，但是又不想关闭 session 怎么办呢，可以输入 <prefix> d，暂时离开。输入打开指定 session 命令又回到了原来的状态。爽~~~

window

创建window

session 可以有多个 window ，你可以在不同的 window 间来回切换处理不同的工作，快捷键是 <prefix> n 。如何创建呢，输入<prefix> c

删除window

<prefix> &

重命名window

<prefix> ,

好像直接贴出了快捷键，其实只要大家理解了三个东西的概念之后，快捷键就是熟练的过程。

搜搜window

还可以在多个 window 之间搜索，真是太贴心了

<prefix> f

Pane

说实话，我认为 pane 很酷，在一个屏幕下可以显示多个面板，比如一个面板写代码，一个运行结果，想想多觉得有用。

水平创建面板

<prefix> %

垂直创建面板

<prefix> "

顺序选择不同的面板

<prefix> o

左右上下切换不同面板

<prefix> { 左或者上面切换

<prefix> } 右或者下面切换

删除面板

<prefix> x

更换面板排版

<prefix> <space> 建议自己尝试看效果

移动面板到window

<prefix> ! 会将面板移动到一个新的 window 里面

移动面板到存在的window

<prefix> :join-pane -t window-name

显示面板编号

<prefix> q

使用手册

列出来的东西毕竟是有限的，如果忘记了快捷键怎么办，

  1. <prefix> ? 看看，多看几遍就熟悉了。
  2. 在终端输入 tmux ，按 <tab> 会显示出命令列表
