关于amd64和386

编译选项中只有这项是不太了解的，具体什么意思呢？我也不知道。 Google 之后，了解的大致的意思，即 amd64 不仅仅局限于 amd 的芯片， amd64 和 Inter64 基本一致，故也可以泛指 Inter 芯片，只不过在某些操作系统上先于 Inter 引入 64 为架构，并且称之为 amd64 ，所以大家习惯上称之为 amd64 和 i386 。同时， x86_64 和 amd64 也是 64 位系统架构
